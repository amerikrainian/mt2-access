# Monster Train 2 Accessibility Installer

Rust installer for Monster Train 2 Accessibility.

This installer is adapted with permission from the SayTheSpire2 installer:
https://github.com/bradjrenshaw/say-the-spire2

Build from this directory with:

```sh
cargo build --release
```

On Windows, `wxdragon` requires Ninja for its native build. The repository release
scripts add Visual Studio 2022 Community's bundled Ninja to `PATH` when it is
available.
